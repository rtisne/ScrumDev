Installation

  • Entrez le script src/bd/cdp_scrum_bdd.sql dans votre gestionnaire MySql.
  • Glissez le dossier /src/web dans votre dossier racine de votre serveur ou créez un host vers ce dossier.
  • Modifiez les valeurs de connexion a la base de données dans le fichier src/web/database_config.php.
