Inscription d'un nouvel utilisateur :

Connexion à la page "signup.php"
Remplir les champs "Nom","Prénom","Email" et "Password"
Appuyer sur le bouton "S'inscrire"

Le nouvel utilisateur est bien inscrit dans la base de données, il peut utiliser son
compte pour se connecter.







Connexion d'un utilisateur inscrit :

Connexion à la page "signin.php"
Remplir les champs "Email" et "Password"
Appuyer sur le bouton "Se connecter".


Une fois connecté l'utilisateur accède à la page listant ses projet








Modification des paramètre du compte d'un utilisateur :

Connexion à la page "listProjects.php"
Cliquer sur le bouton représenté par le nom et prénom de l'utilisateur
Cliquer sur "Paramètre du compte" dans le menu déroulant
Connexion automatique à la page "myAccount.php"
Remplir les champs "Nom","Prénom","Email" et "Password"
Appuyer sur le bouton "Modifier"

Les données de l'utilisateur ont bien été modifié







Déconnexion d'un utilisateur :

Connexion à la page "listProjects.php"
Cliquer sur le bouton représenté par le nom et prénom de l'utilisateur
Cliquer sur "Deconnexion" dans le menu déroulant

l'utilisateur a été déconnecté du site 






Création d'un nouveau projet :

Connexion à la page "listProjects.php"
Appuyer sur le bouton "Creer un nouveau projet"
Remplir les champs "Nom" et "Description"
Ajouter un ou plusieur membre au projet :
	- Rechercher l'utilisateur par nom ou par email dans la barre de recherche "membres"
	- Cliquer sur l'utilisateur correspondant dans la liste déroulante qui s'affiche
Ajouter un product owner au projet :
	- Cliquer sur la flèche grise pour faire dérouler la liste
	- Cliquer sur un des membres dans la liste 
Appuyer sur le bouton "Valider"

Le projet à correctement été ajouté






Modifier les paramètres d'un projet dont l'utilisateur est le créateur :

Connexion à la page "listProjects.php"
Cliquer sur le titre d'un projet à modifier possèdant une icone "Clef à Mollet"
Remplir les champs "Nom" et "Description"
Ajouter un ou plusieur membre au projet :
	- Rechercher l'utilisateur par nom ou par email dans la barre de recherche "membres"
	- Cliquer sur l'utilisateur correspondant dans la liste déroulante qui s'affiche
Ajouter un product owner au projet :
	- Cliquer sur la flèche grise pour faire dérouler la liste
	- Cliquer sur un des membres dans la liste 
Appuyer sur le bouton "Valider"

Le projet à correctement été modifié
